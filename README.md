- Vite

- React

## Development

Requires Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Deployment

Deployed to GitHub Pages via the production build:

```sh
npm run build
```
